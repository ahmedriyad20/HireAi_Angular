## 🎬 Animation Timeline

### Page Load Sequence:
```
0ms    ├─ Profile avatar animates in (slide up)
100ms  ├─ Profile text animates in (slide up delayed)
200ms  ├─ Tab navigation animates in (slide up)
300ms  ├─ Cards fade in sequentially:
       │  ├─ About Me card (0ms)
       │  ├─ Personal Info card (100ms)
       │  └─ Resume card (200ms)

Skills Tab Load:
0ms    ├─ Skills header fades in
100ms  ├─ Skill cards scale in with staggered delays:
       │  ├─ Card 1: 0ms
       │  ├─ Card 2: 50ms
       │  ├─ Card 3: 100ms
       │  ├─ Card 4: 150ms
       │  ├─ Card 5: 200ms
       │  └─ Card 6: 250ms
       │
       ├─ Progress bars animate fill:
       │  └─ Simultaneous with skill cards (800ms duration)

Activity Tab Load:
0ms    ├─ Activity items slide right with delays:
       │  ├─ Account Created: 0ms
       │  ├─ Last Login: 100ms
       │  ├─ Skills Added: 200ms
       │  └─ Account Status: 300ms
```
